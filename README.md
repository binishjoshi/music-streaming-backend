## Description

Backend for my final year project, Sangeet Music Streaming, using [Nest](https://github.com/nestjs/nest) framework.

### Environment variables

```
EXPOSE_PORT=
APP_PORT=

POSTGRES_USER=
POSTGRES_DB=
POSTGRES_PASSWORD=
POSTGRES_HOST=
POSTGRES_PORT=
PGDATA=
PG_DOCKER_VOLUME=

COOKIE_KEY=
```

### Running the app

```bash
docker compose up
```
